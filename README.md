Usage
---
- 从 [Releases](https://github.com/mos9527/zero-patch-zhs/releases) 中下载打包后补丁
- 解压到游戏根目录
- Have fun!

Build
---
- Clone 本 Repo
- 运行 `build.ps1` 即可
