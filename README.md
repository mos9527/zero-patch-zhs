# Zero Patch 简中计划
原项目：https://github.com/CommitteeOfZero/zero-patch

Features
---
适用于 Steam 版本 1.0.0.0
- 新简中字库（字体：Noto Sans CJK JP）；带字体描边修正
- PlayStation (Dualshock/Dualsense) 键位图标 （移植自 PS3 日版镜像 BLJM61303）
- 跳过启动器 （Launcher.exe）
